# Radiant - AI-Powered Note-Taking App

A lightweight Electron-based desktop application for capturing thoughts and generating AI-powered narratives.

## Tech Stack
- Electron 39+
- Svelte 5
- TypeScript
- Vite
- SQLite (better-sqlite3)
- Markdown with YAML frontmatter

## Quick Start
```bash
npm install
npm start
```
